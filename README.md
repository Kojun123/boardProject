프로젝트 소개
==
명칭 : BoardProject   
개발 인원 : 1명   
개발 기간 : 1월 8일 ~ 2월 7일   
한줄 소개 : 웹 게시판의 작성에 필요한 기술을 공부하고자 하여 직접 만들어본 첫 게시판입니다.   

프로젝트 기능
==
게시글 : 글 전체 조회, 상세 조회, 글 작성, 수정, 삭제, 조회수, 페이징, (제목,내용,작성자)검색   
댓글 : 글 상세 조회 시 댓글 조회, 댓글 작성, 수정, 삭제   
유저 : 유효성 검사 및 중복 검사 회원가입, 로그인
